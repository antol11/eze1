# Soluzione: Traccia Laboratorio EZE1 - Nome Cognome (Matricola)

## 1. Descrizione
Questo repository contiene la soluzione completa della traccia di laboratorio, inclusi:

- Diagrammi UML (classi e sequence) nella cartella `uml/`
- Modello Python in `src/`
- Test automatici in `tests/`

La soluzione è strutturata per essere chiara, facilmente eseguibile e verificabile tramite test.
